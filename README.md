# iOS Swift Style Guide - Gemini Agent Skill

This repository contains the Gemini CLI Agent Skill for our iOS team's strict Swift coding conventions.

## How to Install

Run the following command in your terminal to install this skill globally on your machine:

```bash
gemini skills install https://github.com/saeidha/ios-swift-style-skill.git

```

*(Note: Replace the URL with the actual Git URL of this repository)*

## How to Use

Once installed, simply ask Gemini CLI to review or refactor a Swift file:
`gemini "Refactor ViewController.swift to match our team's code style"`
